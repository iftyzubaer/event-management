###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {

sum(1, "2");
```

- C: `"12"`
- D: `3`

###### 4. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sayHi(name) {
  return `Hi there, ${name}`;
}

console.log(sayHi());
```

- A: `Hi there,`
- B: `Hi there, undefined`
- C: `Hi there, null`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: B: `Hi there, undefined`

<i>the `sayHi` function expects a `name` parameter to be passed when it's called. When you call `sayHi()` without passing any arguments, `name` will be `undefined` within the function.</i>

</p>
</details>

###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let count = 0;
const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C: 3

<i>The `forEach` loop iterates through the `nums` array and increments the `count` variable when `num` is truthy. In this case, all the numbers in the `nums` array are truthy (except for 0), so the count should increase for each element.</i>

</p>
</details>
